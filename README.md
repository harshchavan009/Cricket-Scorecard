# 🏏 IPL Cricket Stats Analyzer

A **Python-based web application** that scrapes IPL cricket statistics, analyzes player and team performance, and generates **interactive visual reports**. The project demonstrates skills in **web scraping, data analysis, and data visualization using Python**.

---

## 📌 Project Overview

The **IPL Stats Analyzer** collects cricket statistics from online sources and converts raw data into meaningful insights using visual dashboards.

Users can explore:

* Player performance statistics
* Team performance trends
* Match insights across seasons
* Interactive charts and reports

This project is designed as a **data analytics and sports analytics dashboard**.

---

## 🚀 Features

* 🔎 Web scraping IPL data from cricket websites
* 📊 Data analysis using **Pandas & NumPy**
* 📈 Interactive visualizations using **Plotly / Matplotlib**
* 👨‍🏏 Player performance dashboard
* 🏏 Team performance analysis
* 📉 Match trend insights
* 🔍 Filters by player, team, and season
* 📑 Visual reports for cricket statistics

## ⚙️ Installation

Clone the repository:

```
git clone https://github.com/yourusername/ipl-stats-analyzer.git
```

Move into the project folder:

```
cd ipl-stats-analyzer
```

Install dependencies:

```
pip install -r requirements.txt
```

---

## ▶️ Run the Application

Start the web application using:

```
streamlit run app.py
```

Open your browser and go to:

```
http://localhost:8501
```
